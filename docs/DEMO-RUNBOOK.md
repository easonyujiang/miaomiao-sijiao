# 妙喵私教 · 比赛 DEMO 演示脚本

**比赛**：抖音创变者大赛 · 赛道二：内容重构——让视频成为你的生活搭子
**时长**：3 分钟
**场景**：用户在 B站观看罗翔讲解「正当防卫」，妙喵插件自动介入，完成 5 关法学案例练习

---

## 一、演示前准备

### 环境检查

```bash
# 1. 启动后端
cd ewa
python run.py
# 确认输出：妙喵私教 ready. DB: data/miaomiao.db

# 2. 加载 Extension
# Chrome → chrome://extensions/ → 开发者模式 → 加载已解压的扩展程序
# 选择 extension/ 目录

# 3. 验证连通性
curl http://localhost:8000/health
# 应返回：{"status":"ok",...}

curl http://localhost:8000/api/ext/health
# 应返回：{"status":"ok","llm_available":...,"subtitle_files":1}
```

### 备选方案

| 场景 | 方案 |
|------|------|
| 无法联网访问 B站 | 使用本地 HTML 文件 + video 标签模拟 |
| LLM API Key 未配置 | 使用离线 FAQ 模式（ext/chat 三层回退） |
| 后端启动失败 | 检查 `.env` 中 LLM Key 配置，或删除 key 用离线模式 |
| Extension 气泡不出现 | 按 F12 查看 Console，过滤 `[妙喵]`，确认视频页 URL 格式 |

### 比赛设备

- **演示电脑**：确保已安装 Chrome、Python 3.11+、git
- **网络**：需要访问 B站（或使用本地离线视频备份）
- **LLM Key**：至少配置一个（Kimi 或 DeepSeek），无 Key 也可演示离线 FAQ
- **备用视频**：提前下载 BV1mJ4m147PG 的离线片段

## 三、演示关键要点

### 必须展示的核心功能

- [x] 🐱 气泡自动出现（视频页检测）
- [x] 📚 课程加载（5 关练习数据）
- [x] ✍️ 答题评分（关键词 + LLM 回退）
- [x] ❌ 纠错反馈（漏掉的要点 + 时间戳跳转）
- [x] ⏪ 播放器跳转（`player.seek()`）
- [x] 😺 猫咪状态变化（idle → watching → analyzing → correcting → celebrating）
- [x] ⭐ 游戏化（星星/小鱼干/成长值）
- [x] 📊 进度追踪（进度条 + 气泡角标）

### 演示技巧

1. **提前打开视频**：先加载好 B站页面，避免等待
2. **准备快捷答案**：在记事本里准备好每关的答案，复制粘贴
3. **控制网速**：如有 LLM Key，提前测试响应速度；如无 Key 用离线模式更稳定
4. **F12 关闭**：不要让观众看到 DevTools
5. **放大面板**：Chrome 可以 Ctrl+滚轮 放大，让观众看清文字

### 如果出问题

| 问题 | 应急方案 |
|------|----------|
| B站打不开 | 用本地 HTML 嵌入视频标签 + 模拟内容 |
| 后端挂了 | 快速重启 `python run.py`，20 秒内恢复 |
| 气泡不出现 | 刷新页面，检查 Console |
| 答题不通过 | 展示纠错功能——失败也是演示的一部分！ |
| LLM 超时 | 切换到离线 FAQ 模式——三层回退自动处理 |
